# User Profile

## Role
Software developer building PAN — a persistent AI operating system.

## Expertise
- Deep knowledge of Android/Kotlin, Node.js, JavaScript
- Comfortable with system architecture, networking, databases
- Familiar with AI/ML concepts, Claude API, voice interfaces

## Work Style
- Voice-first interaction — often dictates via phone
- Expects autonomous execution — "just do it"
- Values session continuity — hates context loss
- Prefers bundled changes over many small PRs
- Wants terse responses, no summaries of what was just done

## Preferences
- Never ask to run manual CLI commands
- Capitalize all UI text properly (Title Case for labels)
- Always check memory/DB before asking the user for context
- Restart services without confirmation
